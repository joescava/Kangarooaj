# Kangaroo AJ S.A.S. - Marketing Site (Next.js)

Estructura base del sitio público de Kangaroo AJ S.A.S. usando Next.js (App Router),
TypeScript y Tailwind CSS, preparado para export estático y despliegue en GitHub Pages.

Estructura principal:

- `apps/marketing`: aplicación Next.js del sitio público.
- `.github/workflows/deploy-marketing.yml`: workflow para construir y desplegar la salida estática a GitHub Pages.
